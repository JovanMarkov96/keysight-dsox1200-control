# Vendor Manuals

This directory holds local copies of Keysight documentation used during development.

PDF and CHM files are excluded from version control (see `.gitignore`).

| `Keysight InfiniiVision DSOX1204 users guide.pdf` | Front-panel operation and hardware specs |
| `Keysight InfiniiVision DSOX1204 users guide 2.pdf` | Alternate revision of user guide |
| `Keysight InfiniiVision 1200 X-Series.pdf` | Product overview |
| `Keysight InfiniiVision 1200 X-Series 2.pdf` | Alternate revision of product overview |
| `InfiniiVision 1000 X-Series Oscilloscopes.pdf` | Related family reference |
| `Declassification of the InfiniiVision x-series oscilloscope.pdf` | Declassification/security guide |
| `9018-07746.chm` / `9018-07746 (1).chm` | Windows Help format documentation |
